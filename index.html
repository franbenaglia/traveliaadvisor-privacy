Privacy Policy (English)

Travel IA Advisor is developed by Francisco Arce Benaglia, Argentina.

Contact: francisco.arcebenaglia@gmail.com

1. Information We Collect

  • Account Information: Email address, name, and profile information when signing in via Google OAuth2 or email/password.
  • Location Data (GPS): Used in real-time to determine nearby points of interest (POIs). Location data is processed temporarily and is not stored permanently on our servers.
  • Chat Queries: User questions related to travel and POIs are processed to provide results.
  • Photos: Photos taken within the app are saved only on the user’s device. They are not uploaded or stored on external servers.
  • Purchases: Credits and subscriptions processed via Google Play Billing. We do not store payment card information.

2. How We Use Information

  • To display nearby POIs using GPS and Geoapify API
  • To respond to user chat queries
  • To authenticate users
  • To manage credits and subscriptions
  • To maintain service functionality and security

3. Third-Party Services

We use the following services necessary for app functionality:

  • Amazon Web Services (AWS) – backend processing
  • Geoapify API – geographic data and POIs
  • Google OAuth2 – authentication
  • Google Play Billing – payments

Location data may be transmitted to Geoapify solely to retrieve nearby POIs.

4. Data Storage

Location data is processed in real-time and not permanently stored.

Photos remain stored locally on the user’s device.

Account information is stored securely on AWS infrastructure.

5. Children

This app is not specifically directed to children under 13. If a parent believes a child has provided personal information without consent, please contact us for removal.

6. Data Deletion

Users may request deletion of their account and associated data by contacting:

francisco.arcebenaglia@gmail.com

7. Changes

We may update this policy from time to time. Changes will be posted on this page.

Política de Privacidad (Español)

Travel IA Advisor es desarrollada por Francisco Arce Benaglia, Argentina.

Contacto: francisco.arcebenaglia@gmail.com

1. Información que recopilamos

  • Información de cuenta: Correo electrónico, nombre y datos de perfil cuando se inicia sesión mediante Google OAuth2 o usuario y contraseña.
  • Datos de ubicación (GPS): Utilizados en tiempo real para determinar lugares de interés cercanos (POIs). No se almacenan de forma permanente en nuestros servidores.
  • Consultas en chat: Las preguntas del usuario se procesan para brindar resultados relacionados con lugares de interés.
  • Fotografías: Las fotos tomadas en la aplicación se guardan únicamente en el dispositivo del usuario. No se suben ni almacenan en servidores externos.
  • Compras: Créditos y suscripciones gestionados por Google Play Billing. No almacenamos datos de tarjeta.

2. Cómo utilizamos la información

  • Para mostrar lugares cercanos utilizando GPS y la API Geoapify
  • Para responder consultas del chat
  • Para autenticar usuarios
  • Para gestionar créditos y suscripciones
  • Para mantener la seguridad y funcionamiento del servicio

3. Servicios de terceros

  • Amazon Web Services (AWS) – procesamiento backend
  • Geoapify API – datos geográficos y POIs
  • Google OAuth2 – autenticación
  • Google Play Billing – pagos

Los datos de ubicación pueden transmitirse a Geoapify únicamente para obtener lugares cercanos.

4. Almacenamiento de datos

La ubicación se procesa en tiempo real y no se almacena permanentemente.

Las fotografías permanecen almacenadas en el dispositivo del usuario.

La información de cuenta se almacena de forma segura en infraestructura AWS.

5. Menores de edad

La aplicación no está dirigida específicamente a menores de 13 años. Si un padre o tutor considera que un menor proporcionó información sin consentimiento, puede contactarnos para su eliminación.

6. Eliminación de datos

Los usuarios pueden solicitar la eliminación de su cuenta y datos asociados escribiendo a:

francisco.arcebenaglia@gmail.com

7. Cambios

Esta política puede actualizarse ocasionalmente. Las modificaciones se publicarán en esta página.
